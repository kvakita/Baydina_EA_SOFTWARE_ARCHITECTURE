# Company Scoring System for branchs
Проектирование системы скоринга филиалов компании-контрагента на основе информационной модели рисков

**Пользователи:** риск-аналитики, администраторы стратегий, наша компания как владелец системы.

**Требования:**

1) Клиент для риск-аналитиков:
   * запуск проверки компании по ID,
   * автоматическое определение структуры компании (филиал/головная),
   * получение данных по филиалу и головной компании,
   * просмотр унифицированного результата скоринга,
   * просмотр объяснения результата,
   * просмотр детализированных данных по каждому уровню компании,
   * просмотр аудита проверки (история шагов, источники данных, применённые правила).

2) Серверная часть (оркестратор + сервис скоринга):
   * определение структуры компании по ID,
   * загрузка данных из внутренних источников (CRM, базы компании),
   * загрузка данных из внешних источников (ЕГРЮЛ, санкционные списки, госреестры),
   * приведение данных филиала и головной компании к единому формату,
   * выполнение правил скоринга по каждому уровню (филиал и головная),
   * формирование унифицированного итогового скоринга,
   * формирование объяснения результата,
   * логирование всех этапов обработки (traceID),
   * выдача результата клиенту.

3) Интерфейс администратора стратегий:
   * управление стратегиями скоринга,
   * добавление новых стратегий,
   * обновление и отключение правил,
   * просмотр истории и*
